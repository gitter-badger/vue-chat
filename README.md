# vue-chat

> 项目主要用vue + VantUI 构建的仿bilbili的webapp。前端代码用Docker部署在Nginx容器中



## 技术栈

vue2 + vue-router + vux + websocket



## 说明

后端用springboot搭建的websocket服务端。因为事件的原因，还有其他项目在尝试，只能抽出空余时间完善。
